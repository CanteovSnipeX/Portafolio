Lo que Aprendí

Aprendí a utilizar las tecnologías de JavaFX para el desarrollo de interfaces gráficas con una complejidad mayor,así como iReport para realizar automáticamente reportes de los datos almacenados en la base de datos desde Java. También Pude mejorar mis habilidades en programación orientada a objetos y la utilización de clases, ya que el proyecto en algunos aspectos nos ponía a prueba a experimentar con nuevas librerías para implementar más funcionalidades.

"En este proyecto fue uno donde más empeño he puesto, ya que me hizo mucha ilusión. 🥰"

Descripcion del Proyecto

Es una aplicación de escritorio con una conexión a una base de datos local de MySQL, en la que se lleva a cabo el control de la información de un restaurante, es una aplicación capaz de gestionar gran parte de la información que se puede llegar a manejar en un restaurante.

Funcionalidades


  • Empresas Afiliadas al restaurante
  • Servicios que ofrese el restaurante
  • Presupuesto que se presupuesta a gastar con cada empresa afiliada
  • Categoria o rango de los Empleados
  • Nuevos Empleado
  • Categoria de los platillos existentes
  • Nuevas creaciones del Chef
  • Productos de recien ingreso
  • Datos del creador del sistema

Funciones generales o que aplican a los aspectos especulados anteriormente: Agregar, Editar, Listar, Eliminar y Generar un reporte>
