Agenda Web

Tecnología

Esta aplicación fue creada con Nodejs - Angular - MongooseDB - Express

Lo que Aprendí

A la hora de la realización de este proyecto mejoré mis conocimientos sobre arreglos y sus respectivas funciones en JavaScript, ya que el proyecto se basa en la utilización de comandos, lo cual hace necesario el uso de arreglos para el manejo de los parámetros del comando, así como también agrega mayor complejidad a las validaciones necesarias para el correcto funcionamiento de la aplicación.

Tambien pude conocer un framework de Front End donde pude aprender como implementar las funcionalidades de un api a una vista en un navegador web

Descripcion del Proyecto

Es una aplicacion web que su objetivo fue darnos a conocer Angular y sus funcionalidades

El proyecto consiste en una agenda web con las funcionalidades de registrarse, loguearse, crear contactos, ver los contactos agregados, diferencia de roles y validaciones de tokens

Este proyecto fue realizado junto al profesor encargado de la materia Josué Noj creditos totales.
