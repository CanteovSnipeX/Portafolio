Calculadora Java

Tecnologías

Este proyecto fue creada en JavaFx y el entorno de desarrollo Netbeans

Lo que Aprendí

Aprendí los conceptos iniciales sobre lo que es el patrón de diseño MVC. También aprendí la utilizar objetos como lo son los botones de la vista en JavaFX para ejecutar diferentes funciones y mostrar el resultado de las mismas nuevamente en la vista. Pude mejorar mis habilidades en programación orientada a objetos y la utilización de clases.

La creación de una calculadora no es del todo fácil, esta requiere de cierta concentración en el uso de estos elementos. Debemos llevar un orden para evitar grandes errores que nos puede costar el código completo.

Descripcion del Proyecto

El proyecto realizado es una calculadora básica, capaz de realizar operaciones aritméticas sobre números reales como sumar, restar, multiplicar, dividir, así como soporta la utilización de radicales, fracciones, porcentajes y exponentes. Además, cuenta con las opciones de borrar todo o solo parte de la operación (C y CE), utilización de decimales, además de contar con validaciones matemáticas de tipo MathError.

Es una calculadora completa que tiene todas las funcionalidades de una calculadora Estándar, tiene un entorno para el usuario muy amigable

Estructura


  • Interfaz intuitiva: La calculadora cuenta con una interfaz totalmente intuitiva para el usuario final.

  • Botones: Muestra un diseño con botones incluyendo desde los números, operaciones aritmetica básicas y complejas, cambio de signo, punto decimal, operación inversa, borrar un solo carácter de derecha a izquierda y limpiar toda la pantalla de la calculadora.

  • Código fuente: el código fuente utiliza funciones para poder realizar todas las operaciones aritméticas.
